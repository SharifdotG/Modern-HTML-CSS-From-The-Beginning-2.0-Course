# Leno Website Project — BEM Practice (Sections 19 & 20)

A multi-page marketing site built during Sections 19 and 20 of the **"Modern HTML & CSS — From The Beginning"** course (Udemy). This project emphasizes the **Block‑Element‑Modifier (BEM)** naming convention for CSS while implementing responsive layouts and interactive components with HTML, CSS, and a small amount of JavaScript.

---

## 🔎 Demo

Live site: <https://sharifdotg-leno-website-project.vercel.app>

---

## 📁 Project structure

- `index.html` — Main landing page
- `details.html` — Secondary page (product/service details)
- `css/styles.css` — Styles using BEM class names
- `js/script.js` (and possibly others) — Basic interactivity scripts
- `images/` — Visual assets

---

## ▶️ Run locally

1. Clone or open this project folder
2. Launch either `index.html` or `details.html` in your browser (or use **Live Server** extension)

No build step or server required; static HTML/CSS/JS only.
