# Lumina Creative Grid - Project (148 - Lumina Creative Grid Refactor)

A small, responsive marketing website (Using CSS Grid) built as one of the projects in the **"Modern HTML & CSS — From The Beginning"** course (Udemy).

---

## 🔎 Demo

Live site: <https://sharifdotg-lumina-creative-website.netlify.app/>

> Deployed on Netlify - use the link above to preview the finished, responsive design.

---

## 📁 Project structure

- `index.html` — Home / portfolio
- `about.html` — About the studio
- `contact.html` — Contact form page
- `css/styles.css` — Main stylesheet with CSS Grid
- `images/` — Photography and assets

---

## ▶️ Run locally

1. Clone and open this folder in your editor
2. Open `index.html` in your browser (or use the **Live Server** VS Code extension)

No build tools or servers required — static HTML/CSS only.
