# Tutor Website Project (Sections 17 & 18)

A responsive, multi-page website showcasing a fictional tutoring service. This project was completed as part of the **"Modern HTML & CSS — From The Beginning"** course on Udemy.

---

## 🔎 Demo

Live site: [*(no deployment yet)*](https://sharifdotg-tutor-website.vercel.app)

---

## 📁 Project structure

- `index.html` — Home page with hero section and service overview
- `about.html` — Information about the tutor/company and mission
- `services.html` — List of tutoring subjects and features
- `contact.html` — Contact form page for inquiries
- `css/styles.css` — Main stylesheet containing layout and responsive rules
- `images/` — Photography, icons, and other visual assets

---

## ▶️ Run locally

1. Clone or open this folder in your editor
2. Open any of the `.html` files directly in your browser
3. (Optional) Install and start **Live Server** for real-time reloads

No build tools are required — this is a static site built with plain HTML and CSS.
